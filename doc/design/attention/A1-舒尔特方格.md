# A1: 舒尔特方格 - 详细设计文档

## 1. 医学背景与原理

### 1.1 医学依据
- **训练目标**：选择性注意、视觉搜索、注意力集中度
- **医学原理**：舒尔特表是经典的注意力评估工具，通过数字搜索训练视觉注意广度和集中能力
- **适用人群**：所有阶段患者
- **训练频次**：建议每天1次，每次5分钟

### 1.2 认知效益
- 提升视觉搜索速度
- 增强注意力集中度
- 改善视觉扫描效率

---

## 2. 需求描述

### 2.1 核心功能
- 显示N×N的格子，每个格子有一个数字
- 用户按顺序从1点击到最大数字
- 记录完成时间和错误次数

### 2.2 难度规则
| 难度 | 格子大小 | 数字范围 | 目标时间 |
|-----|---------|---------|---------|
| 1 | 3×3 | 1-9 | 30秒 |
| 2 | 4×4 | 1-16 | 45秒 |
| 3 | 5×5 | 1-25 | 60秒 |
| 4 | 5×5 | 1-25 | 50秒 |
| 5 | 6×6 | 1-36 | 90秒 |
| 6 | 6×6 | 1-36 | 75秒 |
| 7 | 7×7 | 1-49 | 120秒 |
| 8 | 7×7 | 1-49 | 100秒 |
| 9 | 8×8 | 1-64 | 150秒 |
| 10 | 9×9 | 1-81 | 180秒 |

### 2.3 计分规则
```
基础分 = 100
时间奖励 = max(0, (目标时间 - 实际时间) × 2)
错误扣分 = 错误次数 × 10
最终得分 = 100 + 时间奖励 - 错误扣分
```

### 4.2 算法
```typescript
// 生成打乱的舒尔特方格
function generateSchulteGrid(size: number): number[] {
  const total = size * size;
  const numbers = Array.from({length: total}, (_, i) => i + 1);
  // Fisher-Yates洗牌
  for (let i = numbers.length - 1; i > 0; i--) {
    const j = Math.floor(Math.random() * (i + 1));
    [numbers[i], numbers[j]] = [numbers[j], numbers[i]];
  }
  return numbers;
}
```
