# M3: 图片顺序记忆 - 详细设计文档

## 1. 医学背景与原理

### 1.1 医学依据
- **训练目标**：序列记忆、视觉空间记忆、时序记忆
- **医学原理**：训练海马体介导的情节记忆功能，通过图像序列帮助患者建立时空关联记忆
- **适用人群**：轻中度阿尔兹海默症患者
- **训练频次**：每周3-4次，每次10分钟

### 1.2 认知效益
- 提升视觉序列记忆能力
- 增强空间位置记忆
- 改善时间顺序感知

---

## 2. 需求描述

### 2.1 核心功能
- 展示一组图片，按顺序依次显示
- 显示一段时间后图片消失
- 用户需要按正确顺序点击图片
- 记录正确排列的图片顺序

### 2.2 难度规则
| 难度 | 图片数量 | 展示时间 | 记忆时间 | 提示 |
|-----|---------|---------|---------|-----|
| 1 | 3张 | 2秒/张 | 3秒 | 3次 |
| 2 | 4张 | 2秒/张 | 3秒 | 3次 |
| 3 | 5张 | 1.5秒/张 | 4秒 | 2次 |
| 4 | 6张 | 1.5秒/张 | 4秒 | 2次 |
| 5 | 7张 | 1秒/张 | 5秒 | 2次 |
| 6 | 8张 | 1秒/张 | 5秒 | 1次 |
| 7 | 9张 | 0.8秒/张 | 6秒 | 1次 |
| 8 | 10张 | 0.8秒/张 | 6秒 | 1次 |
| 9 | 12张 | 0.6秒/张 | 8秒 | 0次 |
| 10 | 15张 | 0.6秒/张 | 8秒 | 0次 |

### 2.3 图片素材库
- 等级1-3：日常用品（杯子、苹果、房子等）
- 等级4-6：动物（猫、狗、鱼等）
- 等级7-8：场景（海边、森林、城市等）
- 等级9-10：复杂图片（多元素场景）

### 2.4 计分规则
```
正确摆放数量 = 正确位置图片数 × 10
位置奖励 = 连续正确位置数 × 5
速度奖励 = max(0, 限制时间 - 用时) × 1
总分 = 基础分 + 位置奖励 + 速度奖励
```

## 4. 技术实现

### 4.1 核心逻辑
```typescript
// 序列展示控制
async function showSequence(images: Image[], displayTime: number) {
  for (const img of images) {
    highlightImage(img.id);
    await sleep(displayTime);
    unhighlightImage(img.id);
    await sleep(200); // 间隔
  }
  // 隐藏原图，显示待选区
  showSelectionArea(images);
}
```
