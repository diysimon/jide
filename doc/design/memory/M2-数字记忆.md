# M2: 数字记忆 - 详细设计文档

## 1. 医学背景与原理

### 1.1 医学依据
- **训练目标**：短期记忆、数字广度、工作记忆
- **医学原理**：数字广度测试（Digit Span）是临床认知评估经典方法，测试大脑对数字信息的编码、存储和提取能力
- **适用人群**：所有阶段阿尔兹海默症患者
- **训练频次**：建议每周2-3次，每次5-10分钟

### 1.2 认知效益
- 提升数字工作记忆容量
- 增强注意力集中能力
- 改善信息处理速度
- 延缓数字认知衰退

## 2. 需求描述

### 2.1 核心功能
- 系统朗读一段数字序列
- 用户需要复述刚才的数字
- 正确则增加数字长度，错误则结束当前轮次
- 记录正确复述的最大数字长度

### 2.2 游戏模式
| 模式 | 说明 |
|-----|------|
| 顺背模式 | 按顺序复述数字 |
| 倒背模式 | 倒序复述数字 |
| 混合模式 | 随机正序或倒序 |

### 2.3 难度规则
| 难度等级 | 初始数字个数 | 最高数字个数 | 展示速度(个/秒) |
|---------|------------|------------|---------------|
| 1 | 2 | 4 | 1.0 |
| 2 | 3 | 5 | 1.0 |
| 3 | 3 | 6 | 0.9 |
| 4 | 4 | 7 | 0.9 |
| 5 | 4 | 8 | 0.8 |
| 6 | 5 | 9 | 0.8 |
| 7 | 5 | 10 | 0.7 |
| 8 | 6 | 11 | 0.7 |
| 9 | 6 | 12 | 0.6 |
| 10 | 7 | 13 | 0.5 |

### 2.4 计分规则
```
得分 = 正确回忆的最大数字个数 × 10
倒背模式得分 × 1.5
连续正确bonus = 连续轮次 × 5
```

### 2.5 自适应规则
- 连续2轮正确 → 数字个数+1
- 连续2轮错误 → 数字个数-1
- 倒背模式错误 → 退回顺背模式

## 4. 技术实现

### 4.1 核心逻辑
```typescript
// 数字序列生成
function generateNumberSequence(length: number): number[] {
  const sequence = [];
  for (let i = 0; i < length; i++) {
    sequence.push(Math.floor(Math.random() * 10));
  }
  return sequence;
}

// 音频播放控制
async function playSequence(numbers: number[], speed: number) {
  const delay = 1000 / speed;
  for (const num of numbers) {
    speakNumber(num);
    await delay(delay);
  }
}
```

### 4.2 数据结构
```typescript
interface DigitSpanRecord {
  id: string;
  userId: string;
  mode: 'forward' | 'backward' | 'mixed';
  maxDigits: number;      // 最大正确数字个数
  currentDigits: number;  // 当前数字个数
  score: number;
  rounds: DigitRound[];
}

interface DigitRound {
  roundNumber: number;
  sequence: number[];
  userInput: number[];
  isCorrect: boolean;
  timeUsed: number;
}
```
