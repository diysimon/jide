# M1: 卡片配对 - 详细设计文档

## 1. 医学背景与原理

### 1.1 医学依据
- **训练目标**：工作记忆、视觉记忆、短期记忆
- **医学原理**：通过重复学习和记忆提取，激活海马体神经元，促进神经突触可塑性
- **适用人群**：早期至中期阿尔兹海默症患者
- **训练频次**：建议每周3-5次，每次10-15分钟

### 1.2 认知效益
- 增强视觉短期记忆容量
- 提升模式识别能力
- 改善注意力持续时间
- 延缓情景记忆衰退

### 1.3 禁忌症
- 严重视力障碍（可启用语音模式）
- 手指精细动作障碍（可改为点击模式）
- 重度认知障碍（建议从简单模式开始）

## 2. 需求描述

### 2.1 核心功能
- 显示一组卡片（偶数张），背面朝上
- 玩家每次点击两张卡片
- 如果配对成功，卡片保持正面显示
- 如果配对失败，卡片翻转回背面
- 配对全部完成后显示完成界面

### 2.2 游戏规则
| 难度等级 | 卡片数量 | 卡片类型 | 时间限制 |
|---------|---------|---------|---------|
| 1 | 4对(8张) | 简单图形 | 无限制 |
| 2 | 6对(12张) | 简单图形 | 无限制 |
| 3 | 8对(16张) | 简单图形 | 120秒 |
| 4 | 10对(20张) | 简单图形+颜色 | 120秒 |
| 5 | 12对(24张) | 简单图形+颜色 | 90秒 |
| 6 | 12对(24张) | 复杂图形 | 90秒 |
| 7 | 16对(32张) | 复杂图形 | 60秒 |
| 8 | 16对(32张) | 复杂图形+文字 | 60秒 |
| 9 | 20对(40张) | 复杂图形+文字 | 45秒 |
| 10 | 20对(40张) | 汉字/成语 | 45秒 |

### 2.3 计分规则
```
基础分 = 卡片对数 × 10
时间奖励 = max(0, (规定时间 - 实际用时) × 2)
提示扣分 = 提示次数 × 5
最终得分 = 基础分 + 时间奖励 - 提示扣分
```

### 2.4 自适应规则
- 连续3次完成时间 < 50%规定时间 → 难度+1
- 连续2次未能完成 → 难度-1
- 使用提示 > 3次 → 下次同难度

### 3.3 卡片图案库
- **Level 1-3**: 简单几何图形（○△□◇☆★♡♠♣）
- **Level 4-5**: 图形+颜色组合
- **Level 6-7**: 复杂图标（动物、水果、天气）
- **Level 8-9**: 图文结合（文字描述图形）
- **Level 10**: 汉字/成语配对

### 3.4 音效设计
- 翻牌声音：轻柔的"嗒"声
- 配对成功：愉快的"叮"声 + 鼓励语
- 配对失败：轻柔的"嘟嘟"声
- 完成全部：欢快的庆祝音乐

### 3.5 辅助功能
- **语音播报**：每个操作都有语音提示
- **大字体模式**：卡片图案放大50%
- **高对比度**：卡片边框加粗，颜色对比度提升
- **色盲模式**：使用形状+颜色双重区分

### 4.3 核心算法
```typescript
// 洗牌算法 - Fisher-Yates
function shuffleCards(pairs: number): Card[] {
  const symbols = getSymbolSet(pairs);
  const cards = [...symbols, ...symbols];
  
  for (let i = cards.length - 1; i > 0; i--) {
    const j = Math.floor(Math.random() * (i + 1));
    [cards[i], cards[j]] = [cards[j], cards[i]];
  }
  
  return cards.map((symbol, index) => ({
    id: index,
    symbol,
    isFlipped: false,
    isMatched: false
  }));
}
```

## 5. 难度自适应

### 5.1 升级条件
| 条件 | 动作 |
|-----|------|
| 连续3次得分 > 90分 | 难度+1 |
| 连续3次完成时间 < 50%限制 | 难度+1 |
| 错误率 < 20%持续5轮 | 难度+1 |

### 5.2 降级条件
| 条件 | 动作 |
|-----|------|
| 连续2次未能完成 | 难度-1 |
| 错误率 > 60% | 难度-1 |
| 使用提示 > 3次/轮 | 难度-1 |

### 5.3 特殊情况处理
- 用户可手动选择难度
- 每日首次训练从上次难度开始
- 每周一重置难度挑战

## 6. 数据分析

### 6.1 记录指标
- 配对正确率 = 正确配对数 / 总尝试数
- 平均用时 = 总用时 / 尝试轮数
- 学习曲线 = 不同难度下的表现趋势

### 6.2 可视化
- 周/月正确率趋势图
- 难度-得分分布图
- 与同龄人对比（匿名）

### 6.3 异常检测
- 连续一周正确率下降 > 20%
- 用时突然大幅增加
- 长时间未训练（>7天）
