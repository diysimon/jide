# M4: 人脸姓名匹配 - 详细设计文档

## 1. 医学背景

**训练目标**：情景记忆、语义记忆、人脸识别  
**医学原理**：训练面孔再认记忆，涉及梭状回面孔区(FFA)和海马体的协同工作  
**适用人群**：轻中度患者，较好

---

## 2. 需求描述

### 2.1 核心功能
- 显示多个人脸照片和对应姓名
- 记忆后，人脸照片移动到新位置
- 用户需要将人脸与正确姓名连线

### 2.2 难度规则
| 难度 | 人脸数量 | 记忆时间 | 干扰 |
|-----|---------|---------|-----|
| 1 | 2人 | 15秒 | 无 |
| 2 | 3人 | 20秒 | 无 |
| 3 | 4人 | 25秒 | 1个干扰 |
| 4 | 5人 | 30秒 | 2个干扰 |
| 5 | 6人 | 35秒 | 2个干扰 |
| 6 | 7人 | 40秒 | 3个干扰 |
| 7 | 8人 | 45秒 | 3个干扰 |
| 8 | 10人 | 50秒 | 4个干扰 |
| 9 | 12人 | 60秒 | 5个干扰 |
| 10 | 15人 | 60秒 | 6个干扰 |

### 2.3 人脸素材
- 使用标准人脸数据库（如CelebA_subset）
- 确保年龄段、性别多样性
- 排除表情极端的照片
