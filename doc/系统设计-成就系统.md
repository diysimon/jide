# Jide 成就系统 - 详细设计文档

## 1. 成就系统概述

成就系统旨在通过游戏化激励机制，提升用户训练动力和长期参与度。

## 4. 技术实现

### 4.1 数据结构
```typescript
interface Achievement {
  id: string;
  type: 'consecutive' | 'breakthrough' | 'mastery' | 'collect' | 'family';
  name: string;
  description: string;
  condition: string;
  reward: number;         // 积分奖励
  icon: string;           // 图标ID
  rarity: 'bronze' | 'silver' | 'gold' | 'diamond';
}

interface UserAchievement {
  userId: string;
  achievementId: string;
  unlockedAt: Date;
  progress?: number;     // 当前进度
}
```

### 4.2 检查逻辑
```typescript
// 成就检查触发
async function checkAchievements(userId: string, trainingResult: TrainingResult) {
  // 检查坚持类
  await checkConsecutiveAchievements(userId);
  
  // 检查突破类
  await checkBreakthroughAchievements(userId, trainingResult);
  
  // 检查能手类
  await checkMasteryAchievements(userId);
  
  // 检查收集类
  await checkCollectionAchievements(userId);
}
```

## 5. 积分系统

### 5.1 积分获取
| 来源 | 积分 |
|-----|-----|
| 完成训练 | 10-50分 |
| 成就解锁 | 20-200分 |
| 连续登录 | 5分/天 |
| 满分奖励 | +20分 |

### 5.2 积分用途
- 成就解锁奖励
- 排行榜展示
- 虚拟奖励兑换（未来功能）
