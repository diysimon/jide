# Jide 数据库设计文档

## 1. 数据库概述

本项目使用SQLite作为本地数据库，支持离线训练数据存储，未来可扩展为PostgreSQL。

---

## 2. 表结构设计

### 2.1 用户表 (users)
```sql
CREATE TABLE users (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  username VARCHAR(50) UNIQUE NOT NULL,
  password_hash VARCHAR(255) NOT NULL,
  nickname VARCHAR(50),
  role VARCHAR(20) DEFAULT 'patient', -- patient/caregiver/doctor/admin
  age INTEGER,
  gender VARCHAR(10),
  disease_stage VARCHAR(20), -- early/middle/late
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  last_login_at TIMESTAMP
);
```

### 2.2 账户关联表 (account_relations)
```sql
CREATE TABLE account_relations (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  patient_id INTEGER NOT NULL,
  caregiver_id INTEGER NOT NULL,
  relation_type VARCHAR(20), -- spouse/child/other
  permission_level VARCHAR(20) DEFAULT 'full',
  status VARCHAR(20) DEFAULT 'pending',
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (patient_id) REFERENCES users(id),
  FOREIGN KEY (caregiver_id) REFERENCES users(id)
);
```

### 2.3 训练记录表 (training_records)
```sql
CREATE TABLE training_records (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  user_id INTEGER NOT NULL,
  training_type VARCHAR(20) NOT NULL, -- M1-M8, A1-A4, L1-L4, V1-V3, E1-E3, P1-P2
  difficulty_level INTEGER DEFAULT 1,
  score INTEGER DEFAULT 0,
  correct_rate DECIMAL(5,2),
  time_used INTEGER, -- 秒
  hint_used INTEGER DEFAULT 0,
  error_count INTEGER DEFAULT 0,
  training_data TEXT, -- JSON格式，存储详细数据
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (user_id) REFERENCES users(id)
);
```

### 2.4 用户进度表 (user_progress)
```sql
CREATE TABLE user_progress (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  user_id INTEGER NOT NULL,
  cognitive_dimension VARCHAR(20), -- memory/attention/logic/language/executive/perception
  current_level INTEGER DEFAULT 1,
  total_training_times INTEGER DEFAULT 0,
  total_duration INTEGER DEFAULT 0, -- 秒
  best_score INTEGER DEFAULT 0,
  last_training_at TIMESTAMP,
  adaptive_status VARCHAR(20), -- normal/up/down
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (user_id) REFERENCES users(id)
);
```

### 2.5 成就表 (achievements)
```sql
CREATE TABLE achievements (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  achievement_id VARCHAR(50) UNIQUE NOT NULL,
  type VARCHAR(20) NOT NULL, -- consecutive/breakthrough/mastery/collect/family
  name VARCHAR(100) NOT NULL,
  description TEXT,
  reward INTEGER DEFAULT 0,
  icon VARCHAR(50),
  rarity VARCHAR(20) -- bronze/silver/gold/diamond
);

CREATE TABLE user_achievements (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  user_id INTEGER NOT NULL,
  achievement_id VARCHAR(50) NOT NULL,
  unlocked_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  progress INTEGER DEFAULT 0,
  FOREIGN KEY (user_id) REFERENCES users(id),
  FOREIGN KEY (achievement_id) REFERENCES achievements(achievement_id)
);
```

### 2.6 设置表 (user_settings)
```sql
CREATE TABLE user_settings (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  user_id INTEGER NOT NULL,
  setting_key VARCHAR(50) NOT NULL,
  setting_value TEXT,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (user_id) REFERENCES users(id)
);
```

### 2.7 提醒设置表 (reminder_settings)
```sql
CREATE TABLE reminder_settings (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  user_id INTEGER NOT NULL,
  reminder_type VARCHAR(20), -- training/absence/achievement
  enabled BOOLEAN DEFAULT TRUE,
  config TEXT, -- JSON格式
  FOREIGN KEY (user_id) REFERENCES users(id)
);
```

## 3. 索引设计

```sql
-- 用户索引
CREATE INDEX idx_users_username ON users(username);
CREATE INDEX idx_users_role ON users(role);

-- 训练记录索引
CREATE INDEX idx_training_user ON training_records(user_id);
CREATE INDEX idx_training_type ON training_records(training_type);
CREATE INDEX idx_training_date ON training_records(created_at);

-- 进度索引
CREATE INDEX idx_progress_user ON user_progress(user_id);
CREATE INDEX idx_progress_dimension ON user_progress(cognitive_dimension);

-- 成就索引
CREATE INDEX idx_user_achievement ON user_achievements(user_id);
```
