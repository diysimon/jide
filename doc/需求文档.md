# Jide - 阿尔兹海默症大脑训练系统

## 项目概述

### 项目背景

阿尔茨海默症（Alzheimer's Disease，简称AD）是一种进行性神经退行性疾病，主要影响老年人的记忆、思维和行为能力。根据世界卫生组织数据，全球约有5500万人患有阿尔茨海默症，且这一数字还在持续增长。

**Jide** 是一个开源的、基于Web技术的大脑训练平台，旨在通过科学设计的认知训练游戏和练习，帮助阿尔兹海默症患者延缓病情发展，保持大脑活跃度，同时为照护者提供辅助工具。

### 项目愿景

> "用科技守护记忆，让爱不被遗忘"

- 为阿尔兹海默症患者提供便捷、友好的大脑训练工具
- 通过持续的认知训练，延缓记忆力衰退
- 减轻照护者的负担，提供数据化的康复追踪
- 建立开源社区，持续优化训练方案

### 目标用户

| 用户群体 | 描述 | 核心需求 |
|---------|------|---------|
| 轻中度阿尔兹海默症患者 | 早期或中期患者，仍有自主操作能力 | 简单易用的训练界面，清晰的视觉反馈 |
| 照护者 | 家属或专业护理人员 | 训练数据追踪，进度管理，操作辅助 |
| 医疗专业人员 | 医生、康复师 | 训练数据导出，康复效果评估 |
| 志愿者/研究人员 | 参与项目开发的贡献者 | 清晰的开发文档，贡献指南 |

## 核心训练模块概览（20+种训练方式）

本系统设计了**6大认知维度**，共**24种训练方式**：

### 1. 记忆力训练（8种）
| 编号 | 训练方式 | 训练目标 | 医学依据 |
|-----|---------|---------|---------|
| M1 | 卡片配对 | 工作记忆、视觉记忆 | 重复学习效应 |
| M2 | 数字记忆 | 短期记忆、数字广度 | 数字广度测试 |
| M3 | 图片顺序记忆 | 序列记忆、视觉空间记忆 | 瑞文推理测试 |
| M4 | 人脸姓名匹配 | 情景记忆、语义记忆 | 面孔再认记忆 |
| M5 | 词组记忆 | 言语记忆、联想记忆 | 词语学习测试 |
| M6 | 地点记忆 | 空间记忆、场景记忆 | 地点定位测试 |
| M7 | 事件排序 | 时序记忆、逻辑记忆 | 事件时间线重构 |
| M8 | 颜色形状关联 | 关联记忆、模式识别 | 多模态记忆训练 |

### 2. 注意力训练（4种）
| 编号 | 训练方式 | 训练目标 | 医学依据 |
|-----|---------|---------|---------|
| A1 | 舒尔特方格 | 选择性注意、视觉搜索 | 注意力集中度训练 |
| A2 | 找不同 | 持续性注意、细节观察 | 视觉辨别能力 |
| A3 | 视觉搜索 | 快速搜索、反应速度 | 视觉注意广度 |
| A4 | 追踪目标 | 分配注意、交替注意 | 多目标追踪能力 |

### 3. 逻辑思维训练（4种）
| 编号 | 训练方式 | 训练目标 | 医学依据 |
|-----|---------|---------|---------|
| L1 | 图形推理 | 抽象推理、模式识别 | 瑞文推理测验 |
| L2 | 数独入门 | 逻辑推理、数感 | 数字逻辑训练 |
| L3 | 序列找规律 | 归纳推理、演绎思维 | 序列推理能力 |
| L4 | 分类游戏 | 抽象分类、概念理解 | 范畴流利度训练 |

### 4. 语言能力训练（3种）
| 编号 | 训练方式 | 训练目标 | 医学依据 |
|-----|---------|---------|---------|
| V1 | 词语接龙 | 语义记忆、词汇检索 | 语言流畅性训练 |
| V2 | 造句练习 | 句法组织、表达组织 | 句子构建能力 |
| V3 | 看图说话 | 描述能力、表达能力 | 图片描述测试 |

### 5. 执行功能训练（3种）
| 编号 | 训练方式 | 训练目标 | 医学依据 |
|-----|---------|---------|---------|
| E1 | 双重任务 | 认知切换、多任务处理 | 双重任务协调训练 |
| E2 | 停止-开始任务 | 抑制控制、反应抑制 | 停止信号任务 |
| E3 | 计划安排 | 计划能力、任务管理 | 日常任务规划 |

### 6. 情绪与感知训练（2种）
| 编号 | 训练方式 | 训练目标 | 医学依据 |
|-----|---------|---------|---------|
| P1 | 音乐感知 | 听觉感知、情绪调节 | 音乐治疗理论 |
| P2 | 颜色情绪匹配 | 情绪识别、感知联想 | 情绪认知训练 |

## 功能需求

### 1. 用户系统

#### 1.1 用户注册与登录
- **手机号/邮箱注册**：支持基础注册流程
- **微信登录**（可选）：快速登录，降低使用门槛
- **照护者账户**：支持创建主账户+附属账户，照护者可辅助操作
- **忘记密码**：通过短信/邮箱重置

#### 1.2 用户资料管理
- 基本信息：姓名、年龄、性别、病程阶段
- 健康档案：确诊时间、用药情况、既往病史（可选填）
- 头像设置：支持拍照或上传照片

#### 1.3 家庭账户体系
- **患者账户**：进行训练的主体
- **照护者账户**：可查看患者数据、辅助操作
- **医生账户**：可查看脱敏数据、导出报告
- **账户关联**：通过邀请码或二维码绑定

### 2. 难度自适应系统

#### 2.1 自适应算法核心
```
输入：用户历史表现数据
  - 连续正确率
  - 完成时间
  - 错误模式分析
  - 疲劳程度

输出：动态难度调整
  - 题目难度等级（1-10级）
  - 时间限制调整
  - 提示频率调整
  - 训练量调整
```

#### 2.2 降级机制
| 连续错误次数 | 系统响应 |
|------------|---------|
| 连续2次错误 | 给予提示，降低10%难度 |
| 连续3次错误 | 暂停询问，确认理解，重新调整难度 |
| 连续5次错误 | 退回上一难度等级，重置学习曲线 |
| 持续低于40%正确率 | 推荐更换训练项目 |

#### 2.3 升级机制
| 表现 | 系统响应 |
|-----|---------|
| 连续5次正确且速度快 | 提示"你很棒"，尝试提升难度 |
| 连续8次正确 | 自动提升一个难度等级 |
| 正确率>85%持续10轮 | 开放更高难度挑战 |

### 3. 数据存储系统

#### 3.1 本地存储（IndexedDB）
- 用户基础信息缓存
- 离线训练数据暂存
- 设置偏好本地化
- 最大存储容量：100MB

#### 3.2 云端存储（用户账户绑定后）
- 训练历史完整记录
- 跨设备数据同步
- 家庭共享数据
- 医疗数据导出

#### 3.3 数据结构设计

**训练记录表**
```sql
training_records (
  id,                      -- 主键
  user_id,                 -- 用户ID
  training_type,           -- 训练类型(M1-L4)
  difficulty_level,        -- 难度等级(1-10)
  score,                   -- 得分
  correct_rate,           -- 正确率
  duration,                -- 用时(秒)
  error_patterns,         -- 错误模式(JSON)
  hint_used,              -- 使用提示次数
  created_at               -- 训练时间
)
```

**用户进度表**
```sql
user_progress (
  user_id,                 -- 用户ID
  cognitive_dimension,     -- 认知维度
  current_level,          -- 当前等级
  total_training_times,   -- 训练总次数
  total_duration,         -- 累计训练时长
  best_score,             -- 最高分
  last_training_at,       -- 最近训练时间
  adaptive_status,       -- 自适应状态
  updated_at              -- 更新时间
)
```

### 4. 成就系统

#### 4.1 成就分类
| 类型 | 条件 | 奖励 |
|-----|------|------|
| **坚持类** | 连续训练7/30/100天 | 徽章+称号 |
| **突破类** | 首次完成某训练/达到某分数 | 徽章+称号 |
| **全能类** | 完成所有训练类型 | 专属徽章 |
| **速度类** | 单轮完成时间排名前10% | 速度达人徽章 |
| **家庭类** | 家庭成员共同训练 | 家庭徽章 |

#### 4.2 成就等级
```
🥉 铜牌成就：初学者徽章
🥈 银牌成就：进阶训练师
🥇 金牌成就：训练大师
💎 钻石成就：认知冠军
👑 皇冠成就：终极王者
```

#### 4.3 成就展示
- 个人主页成就墙
- 训练结束弹窗展示
- 成就解锁动画效果
- 成就排行榜（可选）

### 5. 家属查看数据系统

#### 5.1 数据分享机制
- **分享码**：生成限时分享码，家属扫码查看
- **账户关联**：直接绑定家属账户
- **隐私控制**：可选择分享的数据范围

#### 5.2 家属端功能
| 功能 | 说明 |
|-----|------|
| 数据面板 | 查看训练数据总览 |
| 趋势图 | 能力变化曲线 |
| 提醒设置 | 设置训练提醒 |
| 鼓励消息 | 发送鼓励语给患者 |
| 异常提醒 | 长时间未训练提醒 |

#### 5.3 数据脱敏
- 医生账户只显示脱敏数据
- 不显示个人隐私信息
- 数据分析基于聚合数据

### 6. 适老化设计

#### 6.1 视觉优化
- 默认大字体（18px基础）
- 高对比度模式（黑底白字/白底黑字）
- 大按钮设计（最小44px点击区域）
- 简洁界面（减少视觉干扰）

#### 6.2 交互优化
- 语音全程播报
- 操作确认提示
- 错误操作允许撤销
- 简化操作流程（最多3步完成一个操作）

#### 6.3 反馈机制
- 每个操作都有明确反馈
- 成功/失败都有语音提示
- 进度实时显示
- 鼓励性语言（"你做得很好！"）
