## 2. 通用组件设计

### 2.1 AppButton 统一按钮
```vue
<template>
  <button
    :class="['app-button', `app-button--${type}`, { 'is-disabled': disabled }]"
    :disabled="disabled"
    @click="handleClick"
  >
    <slot></slot>
  </button>
</template>

<script setup>
defineProps({
  type: { type: String, default: 'primary' }, // primary/secondary/warning
  disabled: { type: Boolean, default: false },
  size: { type: String, default: 'large' } // large/medium/small
})
</script>

<style>
.app-button {
  /* 大触摸区域 */
  min-height: 48px;
  min-width: 120px;
  padding: 12px 24px;
  font-size: 18px;
  border-radius: 12px;
}
</style>
```
